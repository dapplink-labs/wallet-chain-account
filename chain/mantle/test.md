## 1.get free

- request
```
grpcurl -plaintext -d '{
  "chain": "Mantle"
}' 127.0.0.1:8189 dapplink.account.WalletAccountService.getFee
```
- response
```
{
  "code": "SUCCESS",
  "msg": "get gas price success",
  "slow_fee": "20000000|0",
  "normal_fee": "20000000|0|*2",
  "fast_fee": "20000000|0|*3"
}

```
